# Discovery Summary: AI Architect Accelerator

## Executive Overview

**Product Vision**: A fast-track, project-based learning platform that transforms experienced Software Engineers (3+ years) into job-ready AI Product Builders and AI Architects in 8-12 weeks through building 7 real-world AI products.

**Core Philosophy**: *Learn by shipping, not by watching.*

**Target Outcome**: Graduates leave with a deployed AI product portfolio and interview-ready skills to land AI Architect, AI/ML Engineer, or AI Product Builder roles at top companies.

## Technical Infrastructure

### Database Schema (Core Tables)

```sql
-- Users
CREATE TABLE users (
    id UUID PRIMARY KEY,
    email VARCHAR(255) UNIQUE NOT NULL,
    name VARCHAR(255),
    experience_years INT,
    current_role VARCHAR(100),
    target_role VARCHAR(100), -- 'ai_architect', 'ml_engineer', 'ai_pm'
    created_at TIMESTAMP DEFAULT NOW()
);

-- Program Progress
CREATE TABLE user_progress (
    user_id UUID REFERENCES users(id),
    sprint_number INT NOT NULL,
    status VARCHAR(20), -- 'not_started', 'in_progress', 'completed'
    started_at TIMESTAMP,
    completed_at TIMESTAMP,
    PRIMARY KEY (user_id, sprint_number)
);

-- Project Submissions
CREATE TABLE project_submissions (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    project_number INT NOT NULL,
    github_repo_url VARCHAR(500),
    deployed_url VARCHAR(500),
    submitted_at TIMESTAMP DEFAULT NOW(),
    review_status VARCHAR(20), -- 'pending', 'reviewed'
    ai_feedback JSONB,
    score DECIMAL(3,2)
);

-- Skill Assessments
CREATE TABLE skill_assessments (
    user_id UUID REFERENCES users(id),
    skill_slug VARCHAR(100),
    level DECIMAL(3,2), -- 0.0 to 1.0
    assessed_via VARCHAR(50), -- 'quiz', 'project', 'interview'
    assessed_at TIMESTAMP DEFAULT NOW(),
    PRIMARY KEY (user_id, skill_slug, assessed_at)
);

-- Interview Sessions
CREATE TABLE interview_sessions (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    problem_slug VARCHAR(100),
    company_style VARCHAR(50),
    duration_seconds INT,
    transcript JSONB,
    diagram_data JSONB,
    overall_score DECIMAL(3,2),
    feedback JSONB,
    created_at TIMESTAMP DEFAULT NOW()
);

-- AI Conversations (for AI Mentor)
CREATE TABLE ai_conversations (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    context_type VARCHAR(50), -- 'mentor', 'interview', 'code_review'
    context_id UUID, -- References sprint, project, or interview
    messages JSONB,
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);
```
