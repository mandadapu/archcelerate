## Quick Start: Automated Demo Recording

### Option 1: Playwright for Automated Recording

```typescript
// scripts/record-demo.ts
import { chromium } from 'playwright'

async function recordDemo() {
  const browser = await chromium.launch({
    headless: false,
    args: ['--window-size=1920,1080']
  })

  const context = await browser.newContext({
    viewport: { width: 1920, height: 1080 },
    recordVideo: {
      dir: 'demos/',
      size: { width: 1920, height: 1080 }
    }
  })

  const page = await context.newPage()

  // Scene 1: Homepage
  await page.goto('http://localhost:3000')
  await page.waitForTimeout(3000)

  // Scene 2: Curriculum
  await page.goto('http://localhost:3000/curriculum')
  await page.waitForTimeout(2000)

  // Scroll slowly
  await page.evaluate(() => {
    window.scrollTo({ top: 1000, behavior: 'smooth' })
  })
  await page.waitForTimeout(3000)

  // Scene 3: Week 1 content
  await page.click('text=Week 1: LLM Fundamentals')
  await page.waitForTimeout(2000)

  // Scroll through content
  await page.evaluate(() => {
    window.scrollTo({ top: 2000, behavior: 'smooth' })
  })
  await page.waitForTimeout(5000)

  // Scene 4: Domains
  await page.goto('http://localhost:3000/domains')
  await page.waitForTimeout(3000)

  // Close
  await context.close()
  await browser.close()
}

recordDemo()
```

Run with:
```bash
npx tsx scripts/record-demo.ts
# Output: demos/video-{timestamp}.webm
```

## Post-Production

### Video Editing Steps (DaVinci Resolve)

1. **Import clips**
   - Screen recordings
   - B-roll (if any)
   - Voiceover audio
   - Background music

2. **Edit timeline**
   - Cut to match script timing
   - Add transitions (0.5s cross-dissolve)
   - Sync voiceover with visuals

3. **Add text overlays**
   - Week labels
   - Metric callouts
   - GitHub URL
   - CTAs

4. **Color grade**
   - Slight contrast boost
   - Saturation +10
   - Consistent look across clips

5. **Audio mixing**
   - Voiceover: -3dB
   - Background music: -20dB
   - Normalize levels

6. **Export settings**
   - Format: MP4 (H.264)
   - Resolution: 1920x1080
   - Frame rate: 30fps
   - Bitrate: 10 Mbps
   - Audio: AAC 192 kbps
